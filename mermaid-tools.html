🔷 Mermaid Tools & Repos

Different types of Mermaid for planning

📦 Core
Mermaid.js
github.com/mermaid-js/mermaid
The main library. Text-to-diagrams. Flowcharts, sequence, class, state, ER, Gantt, and more.
flowchart sequence class state ER gantt
Mermaid Live Editor
mermaid.live
Online editor. Write mermaid, see it render instantly. Export as PNG, SVG, PDF.
🔧 Extensions & Tools
beautiful-mermaid
github.com/lukilabs/beautiful-mermaid
ASCII rendering of mermaid diagrams. Export to text/console. Pretty!
vscode-mermAId
github.com/microsoft/vscode-mermAId
VS Code extension. AI-powered diagram generation using Copilot. Type what you want, it draws.
Mermaid Chart
mermaid.chart
Commercial version. Team collaboration, templates, cloud storage.
📊 Diagram Types
flowchart - Process flows, workflows
sequence - How things talk to each other
class - Object-oriented classes
state - State machines
ER - Database schemas
gantt - Project timelines
pie - Pie charts
gitgraph - Git history visualization
requirement - Requirements diagrams
journey - User journey maps
💡 For Your Projects
Planning a feature: flowchart
API calls: sequence diagram
Database: ER diagram
Timeline: gantt chart
User flow: journey
Architecture: flowchart or C4
