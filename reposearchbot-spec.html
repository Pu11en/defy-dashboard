📋 RepoResearchBot Spec

OpenClaw instance for Discord - GitHub Repo Research

🏗️ Architecture

💬 Discord Channel
→
📦 OpenClaw Container
→
🐙 GitHub API
→
📋 Links + Summary

🎯 What It Does

Reads spec/idea from Discord channel
Searches GitHub for relevant repos
Returns clickable links (formatted for Discord)
Summarizes each repo (stars, forks, description)
Later: clone, analyze, customize repos

👤 User Flow

1. User types in Discord
→
2. OpenClaw receives
→
3. Searches GitHub
→
4. Returns formatted links

🛠️ Tech Stack

OpenClaw (the gateway)
Discord channel as input
GitHub API for search
Web search for research
MiniMax/Grok for understanding

📦 Phase 1: Basic Setup

✅ OpenClaw running in Docker
✅ Connected to Discord
✅ Responds to channel messages
✅ Searches GitHub API
✅ Returns formatted links

🚀 Phase 2: Enhanced Research

Analyze repo code structure
Generate summaries
Deep search capabilities
Filter by language, stars
